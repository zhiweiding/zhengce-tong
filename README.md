# 政策通 — 办事指南白话版

把复杂的政策办事流程，用人话讲清楚。

每一篇指南都包含：
- **一句话说清楚**这件事是什么
- 需要什么**材料**
- 去**哪里**办  
- **多久**能办好
- **常见问题** FAQ

## 内容

| 分类 | 指南 |
|------|------|
| 社保 | 社保跨省转移、失业金申领、灵活就业社保缴纳 |
| 医保 | 异地就医报销、门诊慢特病报销 |
| 公积金 | 公积金提取（租房/买房/还贷/离职） |

## 技术

纯静态 HTML/CSS/JS，部署在 [Gitee Pages](https://gitee.com)。

## 声明

内容仅供参考，具体政策请以当地官方机构最新规定为准。
